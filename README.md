# Base DEX Aggregator

Base ağı üzerinde Uniswap V3 ve Aerodrome'u karşılaştıran DEX Aggregator.

## Klasör Yapısı

- `frontend/` → Next.js web sitesi (RainbowKit + wagmi)
- `contracts/` → Solidity akıllı kontrat
- `backend/` → Node.js fiyat API'si

## Çalıştırma

### Backend
```
cd backend
node index.js
```

### Frontend
```
cd frontend
npm run dev
```
